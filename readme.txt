Allows the deletion of spam/unapproved comments from an entire WordPress multisite network

== Description ==
This plugin allows you to delete all spam and/or unapproved comments within a multisite installation, without having to visit each individual site.

This process can be somewhat resource-intensive, so it is not recommended that you perform this action during peak traffic hours.

== Installation ==
1. Upload the multisite-comment-management folder to wp-content/plugins
1. Network-activate the Multisite Comment Management plugin
1. Visit Sites -> Multisite Comment Management in the network administration area
